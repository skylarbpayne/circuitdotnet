# API reference

Circuit publishes five package-level API surfaces:

- `CircuitDotNet` — primary C# façade.
- `CircuitDotNet.FSharp` — F#-first helpers and the lightweight `circuit` program DSL.
- `CircuitDotNet.Core` — runtime-neutral contracts and workflow primitives.
- `CircuitDotNet.MicrosoftAgentFramework` — the Microsoft Agent Framework adapter, DI helpers, and OpenTelemetry observer.
- `CircuitDotNet.Testing` — scripted runtimes and assertion helpers for deterministic tests.
