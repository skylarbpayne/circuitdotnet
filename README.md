# Circuit

Circuit is a typed composition library for agentic .NET.

## Packages

- `CircuitDotNet` — planned primary C# entry point.
- `CircuitDotNet.FSharp` — planned primary F# entry point.
- `CircuitDotNet.Core` — shared contracts and core types; not the recommended entry point for most consumers.
- `CircuitDotNet.MicrosoftAgentFramework` — runtime adapter package skeleton.
- `CircuitDotNet.Testing` — testing helpers package skeleton.

## Status

This repository currently contains the foundational solution, package metadata, and stable core contracts for later stacked PRs.
